הכנת המצגת ועיצובה

א. כתיבה ויצוג חזותי
קחו את התרשים שהכנתם למצגת ואת דפי ה"מסכים" והכינו כל מסך על פי
התכנון:
ביחרו למסכי המצגת עיצוב רקעים מתאים. רקע.
תנו לכל מסך כותרת קצרה ובעלת משמעות. כותרות.
הקלידו במסך את הטקסט, בהתאם לרצף שתכננתם: ניסוח.
רישמו בו כמה שפחות מילים )הסתפקו במשפטים קצרים,
נקודות לדיון וכו'(, הקפידו על שפה נכונה וברורה. היכן שניתן
היעזרו - לחסוך במילים, עדיף להציג את הרעיונות בצורה חזותית
ב"תבניות הכתיבה" שלפניכם!
שלבו את התמונות והיצוגים החזותיים שבחרתם מתיק הפרויקט. יצוג חזותי.
הכינו את יתר אמצעי ההמחשה שבחרתם ושלבו אותם אמצעי המחשה.
במסכים: צלילים, תנועה, וידאו.
הכינו מסך פתיחה למצגת: ביחרו לו רקע מתאים, תנו מסך פתיחה ושמות.
למצגת כותרת קצרה ומושכת והוסיפו את שמות כל חברי הצוות, הכיתה והתאריך.
זה המקום לומר תודה לכל האנשים - כדאי להכין גם מסך "תודות" תודות.
שעזרו לכם!
בידקו שוב שרצף הרעיונות שכתבתם מוצג ברצף הגיוני, משוב נוסף.
שהכותרות והנקודות שרציתם להדגיש מנוסחות בצורה ברורה ומדויקת.
בקשו גם מהמורה לבדוק את התכנים והניסוחים. תקנו ושפרו על פי הצורך. -

ב. עריכה ועיצוב
נצלו את הכשרונות האמנותיים בצוות ועצבו את המצגת בצורה מקורית ונעימה
לעין. במידת הצורך, כדאי להיוועץ במורה למחשבים:
עצבו את הרקעים למסכים. רקע.
ביחרו בגודל אות המתאים לקריאה נוחה. כתב.
השתמשו בצבעים מנוגדים לצורך הדגשה. התאימו את צבע והדגשה.
הצבעים לרקע, כך שיהיו מספיק בולטים עליו.
הגדירו את אופן התחלפות המסכים )אפקטים( וקיבעו את משך אפקטים.
הזמן לצפייה בכל מסך. יחד עם זאת, זיכרו שמטרת המצגת לשתף את
הקהל בידע החדש שרכשתם ולא להציג את "פלאי הטכנולוגיה".

ג. הכנות לתצוגה
ארגנו את שולחן התצוגה, המחשב ותוכנת המצגות. בידקו האם החיבור
לחשמל תקין.
התקינו את המצגת במחשב ובידקו שאכן היא עובדת.
- אם בכוונתכם להשאיר את המצגת פועלת באופן אוטומאטי בתצוגה
כיתבו ליד המחשב הוראות כיצד להגיע למסך הפתיחה ולהתחיל בהפעלתה.
בנוסף, הגדירו את המצגת כך שתתחיל מעצמה לבד, לאחר השיקופית
האחרונה.
היעזרו במישהו אחר, - בידקו האם נתתם מספיק זמן לקריאת כל מסך
שישב ליד המצגת ויתנסה בקריאה.
